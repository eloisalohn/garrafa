R:Sistema Operacional 

 Um sistema operacional é um software, ou um conjunto de softwares, cuja função é gerenciar e gerenciar todos os recursos presentes em um sistema. Isso varia de componentes de hardware e sistemas de arquivos a programas de terceiros.Um sistema operacional pode ser definido como um conjunto de programas projetados especificamente para realizar diversas tarefas, inclusive atuando como um intermediário entre o usuário e o computador. O sistema operacional também tem a função de gerenciar todos os periféricos do computador. O Windows é um sistema operacional GUI multitarefa. Isso significa que podemos trabalhar com vários programas ao mesmo tempo. Por exemplo, podemos ouvir um CD inserido em um CD-ROM enquanto escrevemos um documento, imprimimos outro e recebemos um fax, tudo ao mesmo tempo.

São as seguintes:
Executar programas de usuário para solucionar seus problemas mais facilmente;
Tornar o computador conveniente ao uso;
Utilizar o hardware de maneira eficiente;
Compartilhar os recursos de um sistema computacional entre os vários usuários.

Periféricos são dispositivos instalados próximos ao computador cuja função é auxiliar a comunicação homem/máquina. Esses dispositivos podem estar na periferia (ao redor) do computador ou dentro do próprio gabinete.

são as seguintes:Gerenciar tarefas e recursos;
Controlam a entrada e saída de softwares;
São multitarefas;
Têm núcleo;
Têm mecanismos para garantir a proteção, administração e realocação dos recursos;
Contam com Kernel.

Um shell é um programa que permite ao usuário acessar os recursos do sistema operacional por meio de um terminal ou interface gráfica. É ele quem cria uma ponte de comunicação entre o kernel do sistema operacional e o usuário/aplicativos/programas.

O kernel, o coração do sistema, é responsável por conectar o software ao hardware. Desta forma, estabelece uma comunicação eficaz entre os recursos do sistema operacional e gerencia suas funções.
